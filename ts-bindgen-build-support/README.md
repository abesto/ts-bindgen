# ts-bindgen-build-support &emsp; [![CI](https://github.com/ratchetdesigns/ts-bindgen/actions/workflows/ci.yml/badge.svg)](https://github.com/ratchetdesigns/ts-bindgen/actions/workflows/ci.yml) [![Latest Version](https://img.shields.io/crates/v/ts-bindgen-build-support.svg)](https://crates.io/crates/ts-bindgen-build-support) [![Rust Documentation](https://docs.rs/ts-bindgen-build-support/badge.svg)](https://docs.rs/ts-bindgen-build-support) ![Crates.io](https://img.shields.io/crates/l/ts-bindgen-build-support)

ts-bindgen-build-support contains some necessary build tooling (primarily macros) for [ts-bindgen](https://github.com/ratchetdesigns/ts-bindgen).

For usage information about ts-bindgen, check out the [ts-bindgen docs](https://docs.rs/ts-bindgen), [online playground](https://ts-bindgen.ratchetdesigns.com), or [repo](https://github.com/ratchetdesigns/ts-bindgen).
